Para compilar se debe ingresar el siguiente comando en la terminal ubicado en el mismo directorio que contiene todo el codigo:

' g++ main.cpp -o "nombre_del_ejecutable" '

Después se puede usar el siguiente comando (Linux) para ejecutar el programa:

' ./"nombre_del_ejecutable" '

Entradas ejemplo:

- Guardar en un archivo la expresión en infijo: "60- 40+108 - 104 +320-133 -45"
- Guardar en un archivo la expresión en prefijo: "- - + - + - 60 40 108 104 320 133 45"
- Guardar en un archivo la expresión en sufijo: "60 40 - 108 + 104 - 320 + 133 - 45 -"
